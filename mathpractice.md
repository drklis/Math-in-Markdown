# Cobb-Douglas Production Function

The CD function is useful in economics: 

$$Y(L,K) = A L^\beta K^\alpha$$

Where
- $Y$ is production/output
- $L$ is labor
- $K$ is capital
- $A$ is tech
- $\alpha$ and $\beta$ are shares contributed
